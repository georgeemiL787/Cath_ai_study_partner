How Neural Networks Learn

From weighted signals to useful representations

A neural network learns by adjusting the strength of connections between simple computational units. Each unit combines its inputs, applies an activation function, and passes the result to the next layer.

The learning loop

  1. Forward pass: transform an input into a prediction.
  2. Loss: measure the difference between prediction and target.
  3. Backpropagation: compute how each weight contributed to the error.
  4. Update: move the weights in the direction that reduces loss.
wnew = wold − η · ∂L/∂w

The learning rate η controls the size of each update. If it is too large, training may become unstable; if it is too small, learning can be painfully slow.
